=== Hide forum attachments from library ===

Contributors: spurge, lakrisgubben, alfreddatakillen
Tags: media, attachments, bbpress, forum
Requires at least: WordPress 3.0
Tested up to: WordPress 3.5.1
Stable tag: 1.0

Hides all bbpress forum attachments from media library

== Description ==

It simply hides all bbpress forum topics and replies attachments from
the media library.

== Installation ==

Download and upload the plugin to your plugins folder. Then activate it in your network administration.
